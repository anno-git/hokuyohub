## 🔄 CI/CD と GitHub Actions

### 自動化されたワークフロー

**GitHub Actions設定:**
```yaml
# .github/workflows/ci.yml
name: CI

on:
  push:
    branches: [ main, develop ]
  pull_request:
    branches: [ main ]

jobs:
  build:
    strategy:
      matrix:
        os: [ubuntu-latest, macos-latest]
        build-type: [Debug, Release]
    
    runs-on: ${{ matrix.os }}
    
    steps:
    - uses: actions/checkout@v4
      with:
        submodules: recursive
    
    - name: Setup dependencies (Ubuntu)
      if: matrix.os == 'ubuntu-latest'
      run: |
        sudo apt update
        sudo apt install -y cmake ninja-build libyaml-cpp-dev
    
    - name: Setup dependencies (macOS)
      if: matrix.os == 'macos-latest'
      run: |
        brew install cmake ninja yaml-cpp
    
    - name: Configure CMake
      run: |
        cmake --preset ${{ matrix.os == 'ubuntu-latest' && 'linux' || 'mac' }}-${{ matrix.build-type == 'Debug' && 'debug' || 'release' }}
    
    - name: Build
      run: |
        cmake --build build/${{ matrix.os == 'ubuntu-latest' && 'linux' || 'darwin' }}-*/
    
    - name: Test
      run: |
        ctest --test-dir build/${{ matrix.os == 'ubuntu-latest' && 'linux' || 'darwin' }}-*/
    
    - name: Upload artifacts
      uses: actions/upload-artifact@v3
      if: matrix.build-type == 'Release'
      with:
        name: hokuyo-hub-${{ matrix.os }}
        path: dist/*/
```

**マルチプラットフォームビルド:**
```yaml
# .github/workflows/release.yml
name: Release

on:
  push:
    tags:
      - 'v*'

jobs:
  build-and-release:
    strategy:
      matrix:
        include:
          - os: ubuntu-latest
            platform: linux-arm64
            preset: linux-release
          - os: macos-latest
            platform: darwin-arm64
            preset: mac-release
          - os: windows-latest
            platform: windows-x64
            preset: windows-release
    
    runs-on: ${{ matrix.os }}
    
    steps:
    - uses: actions/checkout@v4
      with:
        submodules: recursive
    
    - name: Build
      run: |
        cmake --preset ${{ matrix.preset }}
        cmake --build build/${{ matrix.platform }}
        cmake --install build/${{ matrix.platform }}
    
    - name: Package
      run: |
        ./scripts/create_release.sh --platform ${{ matrix.platform }}
    
    - name: Upload to Release
      uses: softprops/action-gh-release@v1
      with:
        files: |
          hokuyo_hub_${{ github.ref_name }}_${{ matrix.platform }}.tar.gz
      env:
        GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```


## 🧪 テスト戦略

### 単体テスト

**Google Test フレームワーク:**
```cpp
// tests/unit/test_sensor_manager.cpp
#include <gtest/gtest.h>
#include "core/sensor_manager.h"

class SensorManagerTest : public ::testing::Test {
protected:
    void SetUp() override {
        manager = std::make_unique<SensorManager>();
    }
    
    std::unique_ptr<SensorManager> manager;
};

TEST_F(SensorManagerTest, AddSensorSuccess) {
    SensorConfig config;
    config.id = "test_sensor";
    config.type = "hokuyo";
    config.ip = "192.168.1.10";
    
    EXPECT_TRUE(manager->addSensor(config));
    EXPECT_EQ(manager->getSensorCount(), 1);
}

TEST_F(SensorManagerTest, RemoveSensorSuccess) {
    // Setup
    SensorConfig config;
    config.id = "test_sensor";
    manager->addSensor(config);
    
    // Test
    EXPECT_TRUE(manager->removeSensor("test_sensor"));
    EXPECT_EQ(manager->getSensorCount(), 0);
}
```
